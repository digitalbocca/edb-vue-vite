## INSTRUÇÕES

Utilize o create-edb-app:

```bash
npm init @digitalbocca/edb-app vue-vite ./caminho/do/projeto
```

- Entre na pasta.
- Instale as dependências.
- Modifique para sua necessidade.

## FERRAMENTAS CONFIGURADAS

- Vite
- Vue 3
- Vitest (coverage, UI, Vue Test Utils)
- Vue Router
- Pinia
- Tailwind CSS
- sass
- eslint
- Vite Vue devtools
- auto-animate
- Tabler Icons
- Vue Use
- Notivue
- vue countup
- axios
